# 📝 React To-Do List

A simple and interactive **To-Do List App** built with React.  
You can add tasks, delete them, and reorder them by moving them **up** or **down**.

---

## 🚀 Features
- ➕ Add new tasks
- ❌ Delete tasks
- ⬆️ Move tasks up
- ⬇️ Move tasks down
- 🔄 Real-time updates using React state

---

## 🛠️ Tech Stack
- **React** (with Hooks)
- **JavaScript (ES6+)**
- **CSS** (basic styling)
